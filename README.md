## plant-petal(使用言語Python)
花画像から花弁の配置を調べる研究を行っています．  

主な使用技術
* 物体検出(YOLOv5: [github](https://github.com/ultralytics/yolov5))
* 前景抽出(grabcut)
* メタ学習(MAML: [github](https://github.com/dragen1860/MAML-Pytorch))

主な使用ライブラリ
* cv2
* pytorch
